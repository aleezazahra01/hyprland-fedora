## Keybindings

| Action                    | Keybinding               |
|----------------------------|-------------------------|
| Open Wofi Menu             | `Super + D`         |
| Launch Terminal            | `Super + Q`         |
| Close Window               | `Super + C`             |
| Toggle Floating            | `Super + F`             |
| Switch Workspace           | `Super + 1..9`          |
| Move Window to Workspace   | `Super + Shift + 1..9`  |
| Screenshot                 | `SHift +PrintScreen`          |
